# Xchange-Live
A minimalist, real-time currency converter that provides instant exchange rate calculations with a focus on speed and accuracy.

Features
Live Rates: Fetches up-to-the-minute data from [ExchangeRateAPI].

Instant Conversion: Updates results as you type.

Responsive Design: Optimized for both desktop and mobile viewing.

Clean UI: Simple, distraction-free interface.

Tech Stack
Language:  Python

Framework: Langchain , OpenAI , Streamlit
